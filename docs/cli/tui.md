# `genesis tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
genesis tui
genesis tui --url ws://127.0.0.1:18789 --token <token>
genesis tui --session main --deliver
```
